# Retail Sales Analytics (SQL + Power BI)

## Overview
End-to-end retail sales analysis using SQL and Power BI to generate insights on sales, profit, customers, and business performance.

## Tools Used
SQL | Power BI | Excel

## Key Analysis
- Total Sales, Profit & Profit Margin
- Category & Region Performance
- Top Profitable and Loss-Making Products
- Customer Segmentation
- Discount Impact on Profit
- Year-wise Sales Growth

## Dashboard
Interactive dashboard built in Power BI showing KPIs, trends, and business insights.

## Dataset
Superstore Sales Dataset
